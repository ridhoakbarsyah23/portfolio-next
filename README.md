# Portfolio Next.js

Portofolio pribadi yang dibangun menggunakan Next.js — situs untuk menampilkan pengalaman, proyek-proyek, dan kemampuan saya secara profesional.

## 📋 Daftar Isi

- [Home]
- [About]
- [Experience]
- [Skills]
- [Projects]
- [Contacts]

## Tentang Proyek

Situs portofolio ini dibuat untuk:

- Memperkenalkan siapa saya dan latar belakang saya.
- Menampilkan proyek‐proyek yang telah saya kerjakan dengan detail.
- Menyediakan cara mudah untuk menghubungi saya.
- Menggunakan teknologi modern agar performa, SEO, dan aksesibilitasnya baik.

## Fitur Utama

- Navigasi cepat antar bagian (tentang, proyek, skill, kontak).
- Halaman proyek dengan detail, gambar, link ke live/demo.
- Responsif dan mendukung tampilan mobile.
- SEO friendly (meta tag, struktur semantik).
- Mode gelap/terang (jika diaktifkan).
- Contact form yang terintegrasi dengan email atau layanan API (atau sekadar mailto jika sederhana).
- (Tambahkan fitur‐khusus Anda: misalnya blog, animasi, integrasi CMS, dll.)

## Teknologi

- Next.js (React framework)
- React (komponen UI)
- Bootstrap
- Vercel/Netlify untuk deploy (atau platform Anda sendiri)
- (Tambahkan yang lainnya: misalnya Sanity/Contentful, Framer Motion, TypeScript, dll.)

## Instalasi & Pengembangan

1. Clone repositori
   ```bash
   git clone https://github.com/ridhoakbarsyah23/portfolio-next.git
   cd portfolio-next
   ```

## Instal dependensi

npm install

## Jalankan mode dev

npm run dev
